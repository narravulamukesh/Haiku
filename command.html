Basic Commands in Haiku OS

File Management

List of common file management commands:

  • ls: Lists the contents of a directory.
  • cp: Copies files or directories.
  • mv: Moves or renames files or directories.
  • rm: Removes files or directories.
  • mkdir: Creates a new directory.
  • rmdir: Removes an empty directory.
  • cat: Concatenates and displays file content.
  • less: Views file content one screen at a time.
  • touch: Updates the access and modification times of a file, or creates a new empty file.
  • find: Searches for files in a directory hierarchy.

Package Management

List of package management commands:

  • pkgman install : Installs a specified package.
  • pkgman update: Updates all installed packages to their latest versions.
  • pkgman uninstall : Removes a specified package.
  • pkgman list: Lists all installed packages.
  • pkgman search : Searches for packages related to the query.

Networking

List of networking commands:

  • ifconfig: Configures network interfaces.
  • ping : Sends ICMP ECHO_REQUEST packets to network hosts.
  • traceroute : Prints the route packets take to the network host.
  • ssh : Connects to a remote host via SSH.
  • wget : Downloads files from the web.
  • curl : Transfers data from or to a server.

System Administration

List of system administration commands:

  • useradd : Adds a new user.
  • passwd : Changes the password for a user.
  • mount : Mounts a file system.
  • umount : Unmounts a file system.
  • df: Displays disk space usage.
  • top: Displays system tasks and resource usage.
  • ps: Displays information about running processes.
  • kill : Terminates a process by its process ID.
  • fdisk: Manipulates disk partition table.
  • reboot: Reboots the system.
  • shutdown: Powers off or reboots the system.
